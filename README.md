# Kalp Hastalığı Risk Tahmin Projesi 🩺

Bu proje, Kaggle Heart Disease veri setini kullanarak makine öğrenmesi ile kalp hastalığı riskini tahmin eder.

## Özellikler
* **Algoritmalar:** KNN, SVM, Random Forest, Logistic Regression.
* **En İyi Model:** Random Forest (%99 başarı).
* **Arayüz:** Streamlit ile geliştirilmiş web arayüzü.

## Kurulum
1. `pip install -r requirements.txt`
2. `streamlit run app.py`
